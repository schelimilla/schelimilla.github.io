Swordle

This is an interactive game
designed based off the popular game
'Wordle'. It utilizes the Turtle python
library for graphic elements.

Game Rules:

The objective of this game is the same as Wordle, except
that users guess six letters words and have 8 tries in total.
Users input 6 letter words, and the boxes turn yellow, green,
or grey depending on whether the letter is present and in
the right position as the hidden word. The box turns yellow if
that letter is in the hidden word but not in the right
position. The box turns green if that letter is in the
hidden word and in the right position. The box turns grey if
that letter is not in the hidden word.
